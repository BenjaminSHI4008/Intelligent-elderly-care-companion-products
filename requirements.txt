# =============================================================================
# [ENV_VARS] 环境变量（可选，用于覆盖 config.yaml 中的敏感项）
# =============================================================================
# 变量名              | 说明                          | 默认值(见 config.yaml) | 必填
# --------------------|-------------------------------|------------------------|------
# DB_PASSWORD         | MySQL root 密码               | 123456                 | 是*
# AI_API_KEY          | 阿里云 DashScope API Key      | —                      | AI 功能需要
# JPUSH_APP_KEY       | 极光推送 AppKey               | —                      | 推送需要
# JPUSH_MASTER_SECRET | 极光推送 Master Secret        | —                      | 推送需要
# AMAP_WEATHER_KEY    | 高德 Web 服务 Key（天气）     | —                      | 天气需要
# JWT_SECRET          | JWT 签名密钥（≥32 字符）      | —                      | 生产必填
# XIAOBAN_CONFIG_PATH | config.yaml 绝对/相对路径     | ../config.yaml         | 否
# SERVER_PORT         | 后端端口（run.sh）            | 8080                   | 否

# 5. Android Studio 打开 XiaoBan → Sync → Run
